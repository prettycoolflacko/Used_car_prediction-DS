Tentang Project Ini

Aplikasi ini membandingkan performa 4 model Machine Learning:

  • Ridge Regression: Model linear sederhana. Cepat, tapi kurang akurat untuk pola kompleks.
  • Decision Tree: Mudah dipahami, tapi cenderung overfitting jika tidak dibatasi.
  • Random Forest: Sangat stabil dan akurat, tetapi membutuhkan resource komputasi besar.
  • HistGradientBoosting: Modern & Efisien. Pilihan terbaik untuk dataset besar.
